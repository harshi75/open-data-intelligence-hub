# Customer Churn Analysis

## Project Overview
This project focuses on analyzing customer churn data to understand why customers leave a service. 
The analysis includes data cleaning, exploratory data analysis (EDA), visualization, and finding important churn patterns.

## Objective
The main objectives of this project are:

- Understand customer behavior
- Identify factors affecting customer churn
- Analyze relationships between customer features and churn
- Generate insights for improving customer retention

## Dataset Details

- Dataset Name: Customer Churn Dataset
- File Format: CSV
- Target Variable: Churn

The dataset contains information about customers including:
- Customer details
- Services used
- Account information
- Payment details
- Churn status

## Data Cleaning

Performed the following preprocessing steps:

- Checked missing values
- Removed duplicate records
- Converted data types
- Handled inconsistent values
- Checked numerical features

## Exploratory Data Analysis (EDA)

The following analysis was performed:

### Customer Churn Distribution
Analyzed the number of customers who stayed and customers who left.

### Feature Analysis
Studied churn based on:

- Customer tenure
- Contract type
- Monthly charges
- Payment method
- Services used

### Correlation Analysis
Analyzed relationships between numerical variables using correlation analysis.

## Visualizations

Created visualizations using:

- Matplotlib
- Seaborn
- Plotly

Charts include:

- Churn distribution chart
- Bar charts
- Customer segment analysis
- Correlation heatmap
- Trend analysis

## Key Findings

Some observations from the analysis:

- Customers with shorter tenure are more likely to churn.
- Contract type has an impact on customer retention.
- Pricing and service-related factors influence churn.

## Business Recommendations

- Improve customer support for high-risk customers.
- Provide retention offers for customers likely to leave.
- Encourage long-term contracts.
- Improve services based on customer feedback.
